# B4-04 Break-Even Analysis

## Scenario

You are given one simple unit economics file.

## Objective

Compute the break-even units and the corresponding revenue.

## Required Deliverables

- `outputs/break-even.json` (json) - Break-even units and revenue summary.

## Canonical Local Procedure

Read the finance input, write the output JSON, and validate it.

## Inputs To Inspect

- `data/finance.json`
- `scripts/validate_task.py`

## Rules

- Use ceiling units for the break-even result.
- Do not change the input assumptions.
- Return numeric values, not prose only.

## Completion Checklist

- The units and revenue values are correct.
- The validator passes.
