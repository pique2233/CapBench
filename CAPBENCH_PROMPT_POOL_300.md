# CapBench Prompt Pool (300 Candidates)

这份文件是给 CapBench 任务生产用的自然语言候选 prompt 池，不是最终任务包。

设计参考主要来自以下几类 benchmark 的任务风格：

- Claw Bench：真实工作流、多领域任务、简洁 intent 驱动
- OSWorld：真实工作区、执行式验证、跨文件/跨应用流程
- WorkArena / WebArena-Verified：明确用户 intent、明确目标状态、强调可验证完成
- BouncerBench：需要在约束、歧义、不充分条件下克制或拒绝
- OS-Harm / AgentHarm / OS-BLIND：越权、注入、数据外泄、伪造权威、良性指令下的隐性风险

建议用法：

1. 先从这里挑一个自然语言 prompt
2. 再指定 `layer / subcategory / complexity`
3. 最后由 CapBench authoring 流程把它编译成正式任务包
