# CapBench Base Prompt Pool (200 Candidates)

这版通用层不再硬绑 `B1-B4` 的固定分法，而是直接做成一个更宽覆盖面的 200 条自然语言任务池，专门用于筛选“真正能测 agent 能力”的题目。

这次设计重点参考了以下 benchmark 的任务覆盖方式：

- ClawsBench：真实知识工作与办公执行任务
- OSWorld：桌面、文件系统、浏览器、跨应用操作
- WebArena / WebArena-Verified / WorkArena：网页导航、表单、后台、知识检索、企业流程
- GAIA：通用助理、多步推理、附件处理、工具使用
- SWE-bench：代码修复、验证、工程交付

因此，下面的 200 条 prompt 会覆盖但不限于这些能力：

- 浏览器检索与网页操作
- 本地网页或前端页面编写
- 操作系统与文件系统处理
- 邮件、日程、文档、表格、汇报
- 审计、核对、复盘、合规
- 数据分析、简单建模、优化
- 代码修复、脚本编写、测试、自动化
- API、配置、日志、服务运行
- PDF、图片、附件、多源信息整合

以下内容是“候选自然语言题目池”，不是正式任务包。
